# 规则库

在这里可以寻找你想要的规则，请善用页面内查找功能。

在已安装应用的情况下点击「添加」即可完成规则的导入。

你可以通过提交 PR 到[这个文件](https://github.com/lz233/TarnhelmDocument/blob/main/docs/rules.md)帮助我们完善规则。贡献过程中，请注意格式和按照拼音首字母排序。
